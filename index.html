My Projects

A collection of some of my work

TheraPet

A Desktop therapy tool and game, coded in Java.
An interactive GUI made using Swing allows users to play with, pet, feed, and talk to your digital pet. JUnit tested, and JSON save files enabled, this app is ready for play.

Pytest Wizard

A VS Code Extension I developed during my internship at WISI America.
Implemented in Typescript, this extension optimized the test development process of our team by 1500%, by allowing users to run Pytests with the click of a button on the VS Code status bar.

Autofonter for Google Docs

This Google Docs extension automatically formats your active document to a desired style.
Implemented in Javascript, this extension allows users to select a font, font size, and line spacing, and apply it to their entire document with the click of a button. Otherwise, it will scan your document for the most common font and apply it to the entire document.

PNG Editor

This collection of C++ programs lets you modify PNGs down to the pixel using pointer maniuplation.
A PNG is turned into a Tree Data Structure and node manipulation lets you apply filters, create effects, or flip your image. Operations are realized through Depth First Search.

Portfolio Website

This site you're on right now! Self taught HTML, CSS and JavaScript has brought it to life, complete with socials and the interactive project descriptions you are currently reading.

Retro RPG Game

Created in Unity, C# scripts and design patterns bring this game to life.
Observer Pattern allows interactions, collisions, and scene switching. 2D Freeform Lighting and Player HUD is currently in the works. (Currently a private repository for development!)
